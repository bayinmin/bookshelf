HTTPS Self‑Signed Cert & Key Commands

1. Basic self‑signed cert + key (OpenSSL)

Command:
openssl req -x509 -newkey rsa:2048 -keyout server.key -out server.crt -days 365 -nodes

2. SAN config file (san.cnf)

Config file contents:
[req]
distinguished_name=req
x509_extensions=v3_req
prompt=no

[req_distinguished_name]
CN=localhost

[v3_req]
subjectAltName=DNS:localhost,IP:127.0.0.1

3. Self‑signed cert + key with SAN

Command:
openssl req -x509 -newkey rsa:2048 -keyout server.key -out server.crt -days 365 -nodes -config san.cnf
